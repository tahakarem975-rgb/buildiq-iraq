## 🔧 Customization Options

### Easy Adjustments
1. **Animation Speed**: Change `duration` values
2. **Colors**: Update RGB values in SVG strokes
3. **Opacity**: Adjust `rgba` alpha values
4. **Intensity**: Modify gradient stops
5. **Lighting**: Edit light ray positions

### Example: Change animation speed
```tsx
// In engineering-blueprint-bg.tsx
transition={{
  duration: 5,  // Change this to speed up/slow down
  repeat: Infinity,
}}
```
